# ❓ FAQ

**Q: Is IRIS a chatbot?**
A: No. It is a Voice-First OS Agent that executes native system commands.

**Q: Why can't I run the agent locally for free?**
A: The UI is free, but the core agent execution is protected to fund development. See [Sponsorship Guide](SPONSORSHIP_GUIDE.md).

**Q: Does it work on Mac/Linux?**
A: Windows is the primary target, but cross-platform support via Electron is active.
