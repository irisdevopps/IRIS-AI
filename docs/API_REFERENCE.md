# 📚 API Reference

Reference for the IPC Channels.

## File System
- `fs:read-file` - Read a text file.
- `fs:write-file` - Write to disk.

## System Actions
- `sys:execute-terminal` - Run a shell command.
- `sys:open-app` - Launch an executable.

## Vision & Audio
- `vision:screen-peel` - Perform OCR on a screen region.
- `audio:set-volume` - Master volume control.
