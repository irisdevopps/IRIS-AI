# 💻 Developer Guide

For contributors and sponsors working on the IRIS codebase.

## Environment Setup
1. Use Node.js v23.
2. Do NOT use purely cosmetic refactoring PRs.
3. If touching 3D (R3F), DO NOT instantiate variables inside `useFrame`.
