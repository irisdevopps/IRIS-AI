# 🎨 Customization

IRIS uses a highly opinionated "Neural OS" aesthetic.

## Styling Rules
- **Framework:** Tailwind CSS v4.
- **Colors:** Deep Backgrounds (`#030303`), Neon Emerald (`#10b981`), Cyan, Purple.
- **Glassmorphism:** `bg-black/40 backdrop-blur-xl border border-white/5`.

## Extending the UI
You can add new widgets to `src/renderer/src/views/` using the Free Tier public repo.
